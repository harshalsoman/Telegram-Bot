# Telegram Bot for Native Indian language speakers
An end-to-end telegram bot which takes audio questions in local Indian languages as input and outputs answers in the same local language as audio.

### Features
- **Multilingual Support**: The bot can translate responses into different languages using Google Translate.
- **Voice Recognition**: Users can interact with the bot using voice commands.
- **Video Processing**: The bot can edit and process video files, including clipping and adding text overlays.
- **Advanced Conversational Capabilities**: Leveraging OpenAI's GPT-3/4, the bot can provide sophisticated and context-aware responses.
- **Telegram Bot Integration**: The bot can be accessed and controlled via a Telegram bot interface.
- **Audio Processing**: The bot can convert, split, and merge audio files.
- **Flexible Chatbot Framework**: Built using LangChain, the bot can handle multiple conversation flows and integrate with various APIs.
